# Deno Fullstack

A minimal fullstack application running on Deno.

**Stack:**

- Hono
- Kysely
- React
- React Router
- Radix Themes
- Vite (for local development)

---

## 🚀 Running Locally

1. `deno i`
2. `docker compose up -d`
3. `deno task dev`

---

## 🛠️ Building

- `deno task build`

Or with Docker:

- `docker build -t deno-fullstack .`
